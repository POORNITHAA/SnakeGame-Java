🚀 Advanced Snake Game (Java Swing)
A modern, high-performance Snake Game built using Java and the Swing library. This version features enhanced 2D graphics, smooth animations, and a polished user interface.

✨ Features
Modern Visuals: Features anti-aliased graphics, radial gradients for food, and a color-tapered snake body.

Fluid Gameplay: Optimized frame updates with adjustable difficulty levels.

Retro Feel, Modern Look: Classic mechanics paired with a sleek, dark-themed grid background.

Pause Functionality: Press 'P' at any time to pause or resume the action.

🕹️ How to Play
Arrow Keys: Move the snake (Up, Down, Left, Right).

Goal: Eat the red glowing food to grow and increase your score.

Avoid: Don't hit the walls or your own tail!

Pause: Press 'P' to toggle the game state.

🛠️ Requirements
Java JDK 11 or higher (Tested on JDK 25).

Any Java-supported IDE (VS Code, IntelliJ, or Eclipse).
